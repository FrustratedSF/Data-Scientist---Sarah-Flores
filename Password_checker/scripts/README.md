# Password Checker

## A simple Python program that checks the strength of a password based on its length. I build this project as part of my continuous python learning journey that also demonstrates basic programming concepts such as loops, conditionals, variables, and string handling.


## Features:
- Detects Weak, Medium, and Strong passwords.
- Continuously asks the user for a password until a strong one is entered.


##Technologies Used:
- Jupyter Notebook
- Python 3


## Concepts Demonstrated:
- while loops
- if/elif/else conditionals
- Variables
- String length using len()
- User input with input()
